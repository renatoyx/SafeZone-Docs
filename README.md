# Safe Zone — Documentação de QA

Repositório destinado à documentação de testes manuais funcionais do projeto 
Safe Zone, desenvolvido na disciplina de Métodos de Desenvolvimento de Software.

## Sobre o Projeto

O Safe Zone é uma aplicação web em formato de dashboard para análise e 
processamento de dados sobre criminalidade e insegurança no Distrito Federal 
e entorno. O objetivo é informar cidadãos, facilitar denúncias de forma menos 
burocrática e auxiliar órgãos competentes na tomada de decisões.

🔗 [Repositório principal do projeto](https://github.com/jj-viana/safe-zone)

## Sobre esta Documentação

Esta documentação foi produzida como parte do desafio prático de QA, com foco 
em testes manuais funcionais no contexto de metodologia ágil (Scrum).

**Tipo de teste:** Manual, Funcional, Caixa-Preta  
**Nível:** Sistema e Aceite  
**Metodologia:** Scrum
